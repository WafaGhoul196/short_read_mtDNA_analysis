# Annotation databases

The curated folder contains annotation files that are generated with the Filtre_data_bases.R.

Files included:
- gnomAD filtered variants
- HelixMTdb modified dataset
- APOGEE data
- MitImpact database
- MitoMAP RNA/tRNA annotations

If you want to use other versions, you can put your files in the raw folder. Be aware that you have to indicate which folder to use in the pipeline.
